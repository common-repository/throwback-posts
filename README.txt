Show your old posts for your new users with Throwback Posts! Previously on this day.

== Description ==

Throwback Posts is a plugin created for blogs and websites with posts. 
Are you a blogger? Does your website have posts? Let’s show your old posts for your new users with Throwback Posts!
This plugin installs in your website a new box with those posts (see screenshots).

If you have any suggestion to the next versions, please let me know.

== Features ==

*   Select Throwback Dates (One year ago, One month ago, etc...)
*   Filter by post categories
*   Exclude specific posts
*   Design Settings (Image icon, colors, titles)
*   Define the maximum number of posts
*   Display Feature Image (if exists)
*   Display Post Excerpt (if exists)


== Installation ==

1. Upload `throwback-posts` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How To Install Throwback Posts Plugin?
Simple. 
Install from your Admin Panel -> Plugins -> Add New.
After install, you only need to activate it.

= Where Are The Settings Of The Plugin?
After activated, the settings will be available on your WordPress Admin Sidebar -> “Throwback Posts”.

= The Box Is Visible By Default?
No. For security security reasons, you need to activate the menu after finish the configuration. 
To active the box, you just need to select “Activate Throwback Posts” option.

= Throwback Posts Is Mobile Friendly?
Sure.
